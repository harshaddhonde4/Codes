## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/your-username/logic-building.git
cd logic-building
```

### Run C Code

```bash
gcc program.c -o program
./program
```

### Run C++ Code

```bash
g++ program.cpp -o program
./program
```

### Run Java Code

```bash
javac Program.java
java Program
```
